Licoes de HTML

Primeiras impressoes

Os ensinamentos iniciais sobre internet nao foram novidade. Os conceitos mais importantes de aula foram principalmente a estrutura do HTML, dividida em text content, markup , referencias e links.

Como HTML funciona

O aprendizado de tags foi bastante legal. Aprendi a fazer negrito , italico , a incluir um link , uma imagem, como esse simbolo do Metallica

Alem de incluir quebras de linha, como fiz para incluir a imagem e agora.
Outro conhecimento legal, mesmo nao entendendo direito, foi o conceito de block elements.

Problemas

Uma dificuldade adicional foi colocar palavras acentuadas. Saiu tudo feio. Imagino que tenha a ver com a codificao UTF-8, mas olho isso com mais detalhes depois.
Arrumei esse detalhe colocando metadata charset no arquivo.

HTML funciona com tags e uma estrutura tipo árvore.
As tags possuem atributos que podem ser imagens, links, etc. Um resumo das tags vista até agora são:
a: junto com o atributo href, posso incluir um link
b: negrito
em: itálico
img: coloca uma imagem com atributos src e alt
p: cria um block de paragrafo
div: cria um block e posso colocar atributos para diferenciar no CSS
