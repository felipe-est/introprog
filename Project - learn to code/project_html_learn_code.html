Wimbledon 2017 - Campanha de Roger Federer

Esta página contém um resumo da campanha do Roger Federer na conquista do título de 2017.
Foi criado por um fã para o projeto inicial de criação de páginas HTML no Udacity.

Partidas da campanha

  1. Primeira rodada
  2. Segunda rodada
  3. Terceira rodada
  4. Oitavas
  5. Quartas
  6. Semi
  7. Final

Primeira rodada - A. Dolgopolov

O primeiro jogo foi contra do Alexandr Dolgopolov. No primeiro set Federer dominou e fechou em 6x3. O ucraniano abandonou quando perdia o segundo set por 3x0.

Segunda rodada - D. Lajovic

Segundo jogo foi contra Dusan Lajovic. Primeiro set vencido por Federer em um tie-break nada disputado (7-0). Segundo e terceito sets totalmente dominados por Federer. (6/3 e 6/2).

Terceira rodada - M. Zverev

Terceiro jogo foi contra Mischa Zverev. Primeiro set vencido por Federer em um tie-break nada disputado (7-3). Segundo e terceito sets totalmente dominados por Federer. (6/4 e 6/4).

Oitavas - G. Dimitrov

Oitavas contra Grigor Dimitrov. Dimitrov tentou, mas foi totalmente dominado pelo jogo superior do suíço. Primeiro set 6/4. Segundo set 6/2. Fechou o terceiro com 6/4.

Quartas - M. Raonic

Quartas contra Milos Raonic. Jogo perigoso, já que Raonic havia vencido no ano anterior. Federer dominou desde o início. 6/4 6/2 e um último set mais disputado, vencido apenas no tie-break(4).

Semi - T. Berdych

Semifinal dura contra Tomas Berdych, com dois tie-breaks vencidos pelo suíço. No último set Federer elevou o nível e quebrou no 6 game e em seguida sacou para a vitória.

Final - M. Cilic

Final contra Marin Cilic. O que era para ser um jogão acabou não ocorrendo devido às bolhas no pé do croata. Jogo fácil apenas para a coroação do rei.
